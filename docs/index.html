Welcome to the C Utility Library Documentation Webpage

Summary

The C Utils library, a.k.a. cutils, is a set of modular, lightweight libraries providing basic, common utilities for C/C++ projects. I am a student and these libraries exist primarily for learning and personal use, not geared towards production or professional use cases. In the name of good practice (and for the small chance someone sees this in the future!) I am still putting effort into its documentation.

As an extra point, this project serves to replace and improve upon one of my older endeavours, zetaml, which was more specifically a maths library for graphics development.

The C Utils library, cutils, is Free and Open Source Software, and the source repository can be found here.
